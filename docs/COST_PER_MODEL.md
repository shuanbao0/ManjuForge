## 8. 注意事项

1. **价格波动:** AI 模型 API 价格 2026 仍在快速调整,本表为 2026-05 快照,以官方控制台为准
2. **节省计划:** 阿里 AI 通用型节省计划可叠加,B 类(图像/视频/语音)折扣 6.2-7.4 折
3. **失败重试:** fal.ai 失败不计费;直连官方多数计费,需在业务代码做幂等保护
4. **批量折扣:** Google Batch API 7.5 折,适合 ManjuForge 分镜图大批量
5. **聚合溢价:** fal.ai 比官方便宜 50-80%,但有并发限制
6. **海外合规:** 国内业务用海外模型需考虑数据出境合规
7. **自部署门槛:** ¥43/部不含 ML 工程师人力(2-3 万/月),月出片 < 50 部不划算
