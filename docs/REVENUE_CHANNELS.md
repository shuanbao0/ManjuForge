## 1. 平台创作者激励(被动 + 流量验证)

最适合 ManjuForge 这种"自产自销"的形态 — 边测产品边赚钱,Token 成本通过激励变相回本。

### 1.1 红果短剧 · AI 短剧扶持计划 ⭐ 首推

- **归属:** 字节系,国内 AI 短剧最大流量入口。
- **2026 现行政策:**
  - AIGC 短剧专项分账(CPM 高于普通短剧约 30%)。
  - 单部爆款流量补贴 ¥3,000 ~ ¥30,000。
  - 入选"AI 创作者计划"再叠加月度保底。
- **结算:** T+30 ~ T+45。
- **适配题材:** 古风 / 玄幻 / 重生 / 萌宠拟人 — AI 视觉容差高的类目最吃香。
- **入口:** 红果短剧 APP → 创作者中心 → AI 创作激励。

### 1.2 抖音 / 即梦创作激励

- **抖音 AIGC 计划:** 发布带 #AIGC 标签的漫剧片段,按播放给现金,千次播放 ¥1 ~ ¥5。
- **即梦激励中心:** 用即梦相关模型生成的作品在抖音/今日头条发布,有专属流量池 + 激励池。
- **结算:** 抖音月结,即梦周结。
- **优势:** 流量基数最大,适合做种子粉丝。

### 1.3 快手 StreamLake / 可灵创作者计划

- **政策:** 用可灵生成的视频在快手发布,享独立分账池 + 优先流量推荐。
- **2026 新增:** 可灵 2.5 创作者激励金,首发奖励 ¥500 ~ ¥5,000 / 部。
- **入口:** 快手创作者服务平台 → AIGC 专区。

### 1.4 B 站(动态漫垂类受众最精准)

- **创作激励:** 千次播放约 ¥3 ~ ¥10(高于抖音)。
- **充电计划:** 粉丝按月订阅,适合长期连载。
- **大会员合作:** UP 主达 1 万粉后可申请,深度内容签约。
- **题材偏好:** 国漫向 / 武侠 / 仙侠 / 悬疑 — 用户付费意愿强。

### 1.5 小红书(种草 + 引流双用)

- **创作者激励:** 千次曝光 ¥0.3 ~ ¥1。
- **真正价值:** 不靠平台分账,靠"AI 漫剧 demo → 私信引流到微信 → C 端商单",转化率显著高于抖音。

## 2. C 端 / 小 B 商单(现金流最快)

直接现金到账,不依赖算法推荐,适合早期验证"我做的东西真有人买吗"。

### 2.1 闲鱼 ⭐ 上手最快

- **市场状况:** "AI 短剧代做""动态漫制作"已是成熟类目,2026 年存量商家上千。
- **客单价:**
  - 30 秒短剧:¥80 ~ ¥300
  - 1 ~ 3 分钟剧情:¥300 ~ ¥1,500
  - 5 分钟以上整集:¥1,500 ~ ¥5,000
- **运营要点:**
  - 标题带关键词:"AI 短剧 / 动态漫 / 漫剧 / 小说推文 / 漫改视频"。
  - 主图用 9 宫格分镜 + 成片二维码。
  - 闲鱼"鱼塘"功能可加入"AI 创作""短视频外包"等同好圈层精准曝光。
- **付款:** 平台担保,T+0 到 T+7。

### 2.2 小红书 + 微信私域

- **打法:** 发"AI 漫剧制作过程"vlog → 评论区/私信留 V → 转化到微信报价。
- **客单:** 比闲鱼高 30% ~ 100%,因为客户认可"创作者本人"而非"店铺"。
- **缺点:** 平台限制导流严,需要养号 1 ~ 2 个月。

### 2.3 猪八戒网 / 一品威客

- **接单类型:** 中小企业宣传片、电商剧情广告、知识付费课程动画。
- **客单:** ¥500 ~ ¥10,000+,毛利较高但流程慢。
- **门槛:** 需要实名认证 + 缴纳保证金 + 完成 5 单才有展示位。

### 2.4 QQ 群 / 微信群(隐性市场,体量惊人)

- **如何找:** QQ 搜"AI 短剧接单""漫剧外包""小说推文";微信通过 1688 / 知乎 / 闲鱼卖家引导加群。
- **特点:** MCN / 小工作室长期发单,单量稳定但单价被压得较低(¥100 ~ ¥500 / 分钟)。
- **适合:** 跑通流程后做"批量代加工"摊薄 Token。

## 3. To B 长期外包(规模化阶段)

跑通前两类、验证质量稳定后,真正的天花板在 B 端 — 单合同月流水 ¥10,000+。

### 3.1 网文平台漫剧化(最大蓝海)

国内主要网文平台 2025–2026 都在批量"小说 → AI 动态漫/AI 短剧"转化,需求方包括:

| 平台 | 归属 | 接洽方式 |
|------|------|---------|
| **番茄小说** | 字节 | 通过红果短剧官方供应商通道申请 |
| **七猫小说** | 七猫文化 | 七猫"内容合作"邮箱 |
| **阅文集团**(起点 / 红袖) | 腾讯 | 阅文版权/IP 合作部门 |
| **中文在线**(汤圆 / 17K) | 中文在线 | 旗下视频化业务"漫之文化" |
| **塔读 / 飞卢** | 完美世界系 | 漫改/视频化外包供应商 |

- **合作模式:** 按集结算(¥800 ~ ¥3,000 / 集)或月度框架(¥30,000 ~ ¥150,000 / 月)。
- **门槛:** 需要拿出 1 部完整 demo + 报供应商资质,初次合作通常先小批量试单。

### 3.2 短剧/MCN 公司

- **代表:** 听花岛(咪蒙系)、九州文化、银色大地、点众科技、麦芽传媒。
- **痛点:** 它们要的是"AI 加快制作环节",不是整集 AI — 比如 AI 出封面、AI 出分镜、AI 出 3 秒 I2V 镜头。
- **接洽:** 招聘网站搜"AI 美术""短剧 AIGC",直接联系制作部 / 美术总监。
- **回报:** 月度固定 ¥15,000 ~ ¥50,000,稳定但需要驻场或半驻场。

### 3.3 To B 平台型客户

- **K12 / 知识付费:** 把课程剧情化(适合萌系、拟人化)。
- **品牌广告:** 电商详情页剧情 / 朋友圈广告短片(¥3,000 ~ ¥20,000 / 条)。
- **政企宣传:** 文旅、博物馆、非遗 — 政策性预算大,但流程长。

## 4. 把 ManjuForge 用到极致 — 渠道与产品力的结合点

ManjuForge 相比通用工具(剪映 AI / 即梦 / 可灵 Web)的差异化卖点:

| 卖点 | 对应渠道的价值 |
|-----|--------------|
| **小说 → 整片端到端** | 网文平台漫剧化(批量化场景刚需) |
| **统一角色/场景一致性** | 闲鱼 C 端最在意的痛点(便宜竞品做不到) |
| **本地优先 + 可控成本** | To B 商单做"按量报价"时利润率高 |
| **多模型混合(DashScope + Kling + Vidu)** | 同一项目可按场景挑性价比最高的模型,见 [COST_HYBRID_PLAN.md](COST_HYBRID_PLAN.md) |

> 💡 建议每跑通一个渠道,反向沉淀一份"渠道 → 题材 → 模型选型 → 成本 → 报价"的最佳实践,逐步形成 SOP。

## 5. 风控与合规提醒

- **AIGC 标识:** 抖音/B 站/红果均要求显著标注"AI 生成",未标注会被限流甚至下架。
- **版权:** 改编网文必须先拿到授权(平台官方供应商通道自带授权);自创剧本最安全。
- **肖像/声音:** 不得使用未授权的真人脸/声;TTS 优先用 CosyVoice、ElevenLabs 等带商用授权的引擎。
- **题材红线:** 涉黄涉赌涉政、未成年人不当内容,任何平台一票否决。
- **结算合规:** B 端单 ¥3,000 以上建议走对公,留税务空间。
