# 跨平台混合方案 — 质量 + 便宜兼得

> **核心思路:** 没有任何一家供应商在所有能力上都是冠军,**最优解永远是混搭** — 每个环节选"质量好 + 单价低 + 有合规折扣"的平台。
>
> **限制条件:** 全部走 API,**不考虑自部署**(运维成本与本场景目标不符)。

## 2. 跨平台选型核心原理

### 2.1 黄金法则

1. **I2V 是 90% 成本** → 砍 I2V 单价是最大杠杆
2. **同质量条件下,聚合平台单价 < 官方直连** → fal.ai 是首选 I2V 路径
3. **fal 失败不计费** → 失败率高的模型用 fal 更安全
4. **LLM / TTS 不超 1%** → 选最强的,用阿里节省计划顺带打折
5. **海报 / 关键帧值得用最强** → Seedream 5.0 / Nano Banana Pro / Kling 3.0
6. **过渡帧用最便宜的** → Seedance 2.0 Fast / Hailuo 02 / wan2.2-i2v-flash
7. **关键帧分级 = 用 20% 预算买 80% 关键质量**

### 2.2 哪些环节用哪家(冠军组合)

| 阶段 | 推荐平台 | 模型 | 折扣路径 |
|------|---------|------|---------|
| LLM | **阿里** | qwen3-max | 节省 8 折 |
| 海报(70 张) | **字节** | Seedream 5.0 | 商务 / 火山节省 |
| 分镜图(1440 张) | **阿里** | wan2.7-image | 节省 7 折 |
| **I2V 关键帧(20%)** | **fal.ai** | Kling 3.0 | fal 单价 |
| **I2V 过渡帧(80%)** | **fal.ai** | Seedance 2.0 Fast | fal 单价 |
| TTS 中文 | **阿里** | cosyvoice-v3-plus | 节省 7 折 |
| TTS 海外 | **Fish Audio / ElevenLabs Pro** | fish-s2 / eleven_multilingual_v2 | PAYG / 套餐摊薄 |

## 7. 关键判断 / FAQ

### Q1: 为什么 I2V 走 fal.ai 而不是直连?

| 模型 | 官方直连 | fal.ai | 节约 |
|------|---------|--------|------|
| Kling 3.0 | ¥0.96/秒(含音画) | **¥0.21/秒** | 78% |
| Veo 3.1 Fast | ¥1.08/秒 | ¥0.72-1.44/秒 | 0-33% |
| Seedance 2.0 Fast | — | **¥0.16/秒** | (无官方直连) |
| Hailuo 02 512P | — | **¥0.12/秒** | (业界最便宜) |

**fal.ai 在 Kling / Hailuo / Seedance 上比官方便宜 50-78%**,且失败不计费。

### Q2: 阿里 AI 通用型节省计划值得开吗?

| 月承诺 | 12 月期折扣 | 单部节约 |
|--------|------------|---------|
| ¥1K-5K | 8.4 折(A)/ 8.2 折(B) | ~16-18% |
| **¥10K-30K** | **8.0 / 7.6 折** ⭐ | **20-24%** |
| ¥50K-100K | 7.6 / 7.2 折 | 24-28% |
| ¥100K-300K | 7.4 / 7.0 折 | 26-30% |

ManjuForge 用阿里只占总成本 20%(I2V 走 fal),所以即使 ¥10K-30K 档,实际单部节约只有 **¥30-50**。
**月出片 ≥ 30 部、阿里部分月支出 ≥ ¥1K 才开始划算**。

### Q3: ElevenLabs Pro $99/月 vs Fish Audio PAYG?

| 维度 | ElevenLabs Pro $99/月 | Fish Audio PAYG |
|------|----------------------|-----------------|
| 月度成本 | 固定 ¥712 | 按使用 |
| 单部成本(48K 字符) | ¥35(摊薄)| ¥0.52 |
| **每月 ≥ 20 部时** | **¥712 < 20×¥0.52 = ¥10.40** 不划算 | ✅ 划算 |
| 质量 | 业界戏剧化基准 | Elo #1 但平淡 |

**结论:** Fish Audio 几乎永远更便宜。ElevenLabs 只在**单部要求戏剧化情感配音**时值。

### Q4: 关键帧分级 I2V 怎么实施?

在 ManjuForge 的 ModelInstance 系统里挂**两个 I2V 实例**:
- `i2v-key`:Kling 3.0 via fal(¥0.21/秒)
- `i2v-transition`:Seedance 2.0 Fast via fal(¥0.16/秒)

业务层标记每个分镜帧:
- 剧情高潮 / 角色特写 / 多镜头 → `i2v-key`
- 过渡 / 走位 / 背景 → `i2v-transition`

通常 **20% 关键 + 80% 过渡**,综合单价 ~¥0.17/秒。

### Q5: 海外发行能省吗?

海外受限于 Veo 3.1 的单价(Veo Fast ¥1.08/秒,Standard ¥2.88/秒),即便 fal.ai 也只能拿到 Fast 价。
**唯一节约路径:** Veo 关键帧 + Seedance 2.0 Fast 过渡(混海外品牌),单部 ¥2,380 是合理底线。

## 9. 关键风险与对冲

| 风险 | 对冲手段 |
|------|---------|
| **fal.ai 限速** | 同时持有 阿里(wan2.7-i2v)/ 火山(Seedance) 直连账号兜底,fal 限速时切流量 |
| **Kling 3.0 via fal 价格上调** | 方案有 Seedance Fast / Hailuo 02 多个降档选择 |
| **阿里节省计划承诺达不到** | 未达成部分超额按按量,不会全部失效 |
| **字节 Seedance 二次涨价**(2026 春已发生) | I2V 主力放 fal,字节只放 Seedream 海报 |
| **Google Veo 价格波动** | Veo 仅用于海外发行场景,占总量小 |
| **失败重试浪费** | 全部 I2V 走 fal(失败不计费),非 fal 走幂等业务代码 |

## 10. 一句话总结

**最强性价比方案:**

> **LLM/T2I/TTS 全部走阿里 + 节省计划 7-8 折,I2V 全部走 fal.ai Seedance 2.0 Fast(¥0.16/秒),海报点缀 Seedream 5.0,精品场景插入 Kling 3.0 via fal**
> → **¥1,160 / 部**,比阿里全旗舰节约 80%,质量 720P 商用足够。

**关键帧分级提升质量:**
> 关键剧情用 Kling 3.0 via fal,过渡用 Seedance 2.0 Fast → **¥2,677 / 部**,质量接近 1080P 旗舰,成本仍是阿里全旗舰一半。
